# Triad Trainer

Uma ferramenta interativa desenvolvida com **Svelte** para auxiliar músicos no estudo, visualização e memorização de tríades no teclado.

**Acesse a aplicação:** [milo-triad-trainer.web.app](https://milo-triad-trainer.web.app/)

## 🚀 Tecnologias

- **Svelte**: Framework para a interface reativa.
- **Vite**: Bundler e ambiente de desenvolvimento rápido.
- **TypeScript**: Tipagem estática para maior segurança.

## ✨ Funcionalidades (Planejadas/Implementadas)

- **Geração de Acordes**: Geração e sorteio aleatório de tríades para prática.
- **Feedback Visual**: Feedback visual imediato sobre acertos e erros.
- **Gabarito**: Visualização de respostas para evitar bloqueios.
- **Diferentes Qualidades**: Prática de tríades Maiores, Menores, Diminutos, Aumentados e Suspensos.
- **Histórico**: Registro persistente de erros segmentado por nota e qualidade, com geração de gráficos informativos para identificação precisa de lacunas de conhecimento.

## 🛠️ Instalação Local

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.

1. Clone o repositório:
   ```bash
   git clone https://github.com/Amyr14/triad-trainer.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
